## 🚀 Быстрый старт

### Локальная установка

1. **Клонируй репозиторий:**

   ```bash
   git clone https://github.com/shaomdev/shaomdev-portfolio.git
   cd shaomdev-portfolio
   ```

2. **Открой в браузере:**
   - Способ 1: Просто открой файл `index.html` в браузере
   - Способ 2: Используй Live Server (VS Code расширение)
   ```bash
   # Если установлен Live Server
   code . # откроется VS Code
   # Нажми правой кнопкой на index.html → Open with Live Server
   ```
