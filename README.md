## 🎯 Ключевые особенности и технические решения

- **Дизайн в стиле Apple:** Глубокий черный фон, строгая типографика, обилие свободного пространства и идеальный баланс контраста.
- **Интерактивная Bento-сетка навыков:** Секция со скиллами реализована в виде современной плитки (Bento Grid) на чистом Flexbox. Каждая карточка плавно подсвечивается при наведении.
- **Побуквенная кинематографичная анимация:** Уникальный ховер-эффект на логотипе `SHAOMDEV` — при наведении буквы плавно окрашиваются в фирменный синий цвет Apple, а снизу волнообразно раскрывается тончайшая неоновая линия.
- **Эффект матового стекла (Glassmorphism):** Фиксированная шапка сайта (`sticky`) использует свойство `backdrop-filter: blur`, создавая премиальный эффект размытия контента, который прокручивается под ней.
- **Параллакс-эффект на карточках проектов:** При наведении курсора карточки работ приподнимаются, а скриншоты внутри них плавно масштабируются (`scale`), создавая ощущение глубины интерфейса.
- **Экосистема проектов:** Все карточки работ ведут на реальные, задеплоенные и полностью рабочие живые страницы (MoGo, Climber Game, Smart Delivery) без битых ссылок и заглушек.
- **100% Адаптивность:** Сайт ювелирно оптимизирован под любые экраны — от огромных мониторов до компактных смартфонов. Элементы навигации на мобилках трансформируются в удобные тач-зоны.

## 🛠 Мой технологический стек

- **JavaScript (ES6+)** — базовые концепции, работа с DOM-деревом, логика веб-приложений (в процессе глубокого изучения и автоматизации интерфейсов).
- **HTML5** — семантическая (`<header>`, `<main>`, `<article>`, `<footer>`) валидная и доступная разметка страницы.
- **CSS3** — кастомные переменные (`:root`), Flexbox, Grid, продвинутые селекторы состояний (`:focus`, `:checked`, `:hover`), медиа-запросы (`@media`) для адаптации.
- **БЭМ (Блок, Элемент, Модификатор)** — строгая и плоская архитектура классов для чистоты и масштабируемости кода.
- **Base64 кодирование** — для интеграции векторной графики напрямую в стили.
- **Git / GitHub** — профессиональный контроль версий и деплой через GitHub Pages.
- **Figma / Photoshop** — уверенная работа с дизайн-макетами, экспорт графики, точное копирование UI-компонентов и пиксель-перфект верстка.
